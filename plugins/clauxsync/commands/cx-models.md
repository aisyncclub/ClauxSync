---
description: "List available Codex models and change the active model."
---

Call the `clauxsync_models` MCP tool to show all available models.

If $ARGUMENTS contains a model ID (e.g. "gpt-5.4"), call `clauxsync_set_model` to switch to that model.

Present the model list to the user and ask which one they'd like to use if no argument was given.
